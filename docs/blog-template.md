## 模板结构说明

| 区域 | 元素 | 说明 |
|------|------|------|
| **顶部** | 语言切换 | `[🌐 English Version](/path/to/en/)` |
| **标题** | 标题 + 描述 | 一句话介绍 |
| **目录** | TOC | 章节编号链接 |
| **引言** | 0.1-0.4 | 定义、优势、受众、Quick Start |
| **正文** | 1-N 章 | 按类别/步骤组织 |
| **结尾** | 总结 + 资源 | 要点回顾 + 延伸阅读 |
| **底部** | 庆祝 + Powered + 更新 + ClustMaps + 版权 | 统一 footer |
